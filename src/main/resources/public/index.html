About

I am current completing the final term of my Bachelor of Software Engineering at @uvic with a specialization in the Theory of Computation. In September I will be joining the web platforms team at @twitter in San Francisco where I will help create and maintain the tools that make Twitter great.

I have an interest in many areas of computer science including CS theory, machine learning, programming languages, randomized alrogithms, HCI and mobile development. I also spend a lot of my free time playing guitar and bass and I hope to be able to pick up some new instruments soon.

Previous Work Experience

Google

June 2013 - August 2013

During my internship a Google, I helped re-write the Chrome OS Recovery Tool as a Chrome Packaged App using AngularJS. This allowed a single UI to be used across all platforms. In order to acomplish this, I also helped add a new API to Chrome that handled writting images of Chrome OS to a user's USB drive.

Wifarer

May 2012 - August 2012

At Wifarer I developed the user interfaces for multiple Android applications and wrote code to retreive information from webservices, store the information in a database on the phone and then display it to the user. I also had an opportunity to extend the functionality of the Wifarer API, making the path finder much more robust.

Broadcom

January 2011 - August 2011

Most of my time at Broadcom was spent developing Android app used to demo Broadcom's core technologies including video calling (VOIP), video streaming (DLNA) and home automation applications. I also helped debug issues with porting new versions of Android to Broadcom's tablets.

Packet Quest

As part of SENG 499 at UVic, I helped develop Packet-Quest, a 2D platfomer game, along with 4 of my classmates. I developed many gameplay elements and created a custom audio system featuring support for fading clips in and out, caching clips, adjusting audio's pitch and allowing any gameplay element to react to changes in the audio.
